# 🍲 Recipe Finder Web App

A modern **Recipe Finder Web Application** built using **HTML, CSS, and JavaScript** that allows users to search, explore, and save recipes with a beautiful UI.

---

## ✨ Features

* 🔍 Search recipes by name
* 🍱 Browse recipes by category
* 🎲 Get random recipes
* 📄 View full recipe details (ingredients + instructions + video)
* ❤️ Save favorite recipes (stored in browser)
* 🌙 Dark mode (with saved preference)
* 🎨 Premium UI (glassmorphism + animations)
* 🍔 Floating animated food background
* 📱 Responsive design

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Glassmorphism + Animations)
* JavaScript (Vanilla JS)
* API: TheMealDB

 

## 🚀 How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/recipe-finder.git
```

2. Open the project folder

3. Run backend (if using Node):

```bash
node app.js
```

4. Open `index.html` using Live Server or browser
